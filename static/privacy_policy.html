Privacy Policy

Outlook Web Right-to-Left is a Chrome extension that enhances the Outlook Web App for Right-to-Left (RTL) language users by applying visual improvements in your browser.

No Data Collection

This extension does not collect, store, or transmit any user data. It does not handle personal or sensitive information in any form.

Compliance with User Data Policy

As the extension does not collect or handle any user data, no special obligations apply under the Chrome Web Store User Data Policy.

Permissions

The extension uses only minimal permissions required to apply visual styling changes to the Outlook Web App. It does not access your content or interact with any servers.

Changes to This Policy

If the extension is updated in the future to handle user data, this privacy policy will be updated accordingly and users will be notified where required.
